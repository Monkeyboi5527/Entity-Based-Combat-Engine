Entity-Based-Combat-Engine

Project Overview:

The Entity-Based Combat Engine is a modular Java-based combat simulation framework designed around entity-driven architecture. The system models combat using entities, components, stats, abilities, and effects instead of rigid inheritance structures.

The engine is designed to be scalable, testable, and reusable for RPG-style or tactical combat systems.

Goal:

-To design and implement a flexible, extensible combat engine that:

-Uses composition over inheritance

-Separates combat logic into systems

-Supports dynamic stats and modifiers

-Allows abilities and status effects to be easily added

-Demonstrates clean architecture and SOLID design principles
